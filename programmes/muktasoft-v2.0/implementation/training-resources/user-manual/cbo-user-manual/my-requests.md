# My Requests

#### My Requests <a href="#docs-internal-guid-a1a3e71a-7fff-9125-33ee-1bb401bff0ce" id="docs-internal-guid-a1a3e71a-7fff-9125-33ee-1bb401bff0ce"></a>

There are basically two types of requests which are raised by CBOs. One is the request to close the project, second is to request to extend the completion period which is known as time extension. Once these requests are raised by CBOs, are sent for approval, and the same can be seen and tracked under My Requests. &#x20;

To view all the requests raised -

1. Click on the My Requests service.
