# User Manual

The MUKTASoft application is a standardized solution offered on the DIGIT Works platform for managing MUKTA related works and activities.&#x20;

## Key Features

The application supports the following vital functions -

1. Project: Create, Search, View, Modify, and Download PDF
2. Estimate: Create, Search, View, Process for Approval, Download PDF
3. Revise Estimate: Create, Search, View, Process for Approval, Download PDF
4. Work Order: Create, Search, View, Process for Approval, Download PDF
5. Time Extension: Create, Search, View, Process for Approval.
6. Measurement Book: Create, Search, View, Process for Approval, Download PDF
7. Muster Roll: Search, View, Process for Approval, Download PDF
8. Billing: Create, Search, View, and Process for Approval
9. Auto Creation: Wage Bill on Muster Roll approval, Supervision Bill on Every Tuesday Night.&#x20;
10. Payment Advice: Search, and download for bill processing
11. Organization: Create, Search, View, and Modify for CBOs/Vendors
12. Wage Seeker: Search, View, and Modify
13. Employee/Users: Create, Search, View, and Modify with role mapping
14. SMS Notifications: Alerts users about completed actions
15. Language Support: English and Odia
